# Alerts

Bootstrap component documentation and examples.

## Alerts

### Description
Alerts component demonstrating various styling and layout patterns.

### Key Bootstrap Classes

- `.alert`
- `.alert-dismissible`
- `.bg-white`
- `.border`
- `.btn-close`
- `.d-flex`
- `.fade`
- `.gap-2`
- `.show`
- `.text-danger`
- `.text-info`
- `.text-success`

### HTML Pattern

```html
<div class="d-flex gap-2">
<div class="alert bg-white border alert-dismissible fade show text-danger" role="alert">
<b>Danger!</b> A simple danger alert.
<button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>
<div class="alert bg-white border alert-dismissible fade show text-success" role="alert">
<b>Success!</b> A simple success alert.
<button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>
<div class="alert bg-white border alert-dismissible fade show text-warning" role="alert">
<b>Warning!</b> A simple warning alert.
<button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>
<div class="alert bg-white border alert-dismissible fade show text-info" role="alert">
<b>Info!</b> A simple info alert.
<button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
```

# Form Wizard

Bootstrap component documentation and examples.

## Default Wizard Style

### Description
Form Wizard component demonstrating various styling and layout patterns.

### Key Bootstrap Classes

- `.col-12`
- `.col-sm-3`
- `.form-control`
- `.form-label`
- `.gx-4`
- `.mb-3`
- `.nav`
- `.nav-item`
- `.nav-link`
- `.num`
- `.row`
- `.tab-content`

### HTML Pattern

```html
<!-- Wizard #1 -->
<div id="smartwizard">
<ul class="nav">
<li class="nav-item">
<a class="nav-link" href="#step-1a">
<div class="num">1</div>
Step Title
</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#step-1b">
<span class="num">2</span>
Step Title
</a>
</li>
<li class="nav-item">
<a class="nav-link" href="#step-1c">
<span class="num">3</span>
Step Title
</a>
</li>
<li class="nav-item">
<a class="nav-link " href="#step-1d">
<span class="num">4</span>
Step Title
</a>
</li>
</ul>
<div class="tab-content">
```

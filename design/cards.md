# Cards

Bootstrap component documentation and examples.

## Card Header

### Description
Cards component demonstrating various styling and layout patterns.

### Key Bootstrap Classes

- `.btn`
- `.btn-warning`
- `.card-img`
- `.img-fluid`
- `.mb-3`
- `.mb-4`
- `.rounded-3`

### HTML Pattern

```html
<div class="card-img">
<img src="assets/images/products/product5.jpg" class="img-fluid rounded-3 mb-3"
alt="Vibe Templates Template" />
</div>
<p class="mb-4">
Some quick example text to build on the card title and
make up the bulk of the card's content.
</p>
<a href="#" class="btn btn-warning">Button</a>
```

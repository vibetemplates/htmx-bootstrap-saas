# Input Tags

## Default

### Description
Input Tags component demonstrating various styling and layout patterns.

### Key Bootstrap Classes

- `.form-control`
- `.form-group`
- `.m-0`

### HTML Pattern

```html
<div class="form-group m-0">
<input type="text" class="form-control" value="Amsterdam,Washington,Sydney,Beijing,Cairo"
data-role="tagsinput">
```
